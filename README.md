# This is repo for Algorithm recording.

## 자주 나오는 유형

- 브루트포스
- 그리디
- 구현/시뮬레이션 (삼성)
- 그래프 이론
- DFS
- BFS
- 정렬
- 이진 탐색
- DP
- 최단경로 (다익스트라?)


## 주요사항

- 백준 기준 실버3 ~ 골드2정도를 막힘없이 풀 수 있으면 베스트.
- 40분정도 안에 못풀면 답지에서 아이디어 얻어서 풀고, 며칠 뒤 다시 풀어보자!
- 컴퓨터는 1초에 약 1억번 연산 가능.
