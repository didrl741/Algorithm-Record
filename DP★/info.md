
## 개념정리

- 상황 : 큰 문제를 작은 문제로 분할할 수 있고, 작은 문제가 반복사용될 때.
- 재귀 : 탑다운 방식. 빠르다.
- 반복문 : 바텀업 방식. 힙 영역을 쓰지 않아 메모리 초과를 막을 수 있다. 느릴 수 있다.
- 푸는 법 : 점화식 + 메모이제이션
- 배열 초기화 함수 두가지 : 
- fill(arr, arr + n + 1, 1000);     -> 모든 수 가능.       algorithm 헤더
- memset(arr, -1, sizeof(arr));     -> 0, -1 만 가능.      cstring 헤더
